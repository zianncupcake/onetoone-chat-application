# onetoone-chat-application

## Description
Full stack 1-1 chat room application - enables creation of private chat rooms with an online user, live notifications for unread messages and when new users join come online, logout to go offline

## Tech stack
Java, Spring Boot, WebSocket, HTML, CSS, Javascript, MongoDB, Mongo Express, Docker
